Detailed Description

ObservableCollection implementation which supports turning off notifications for mass updates through the BeginMassUpdate method.

var coll = new OptimizedObservableCollection<string>();
...
using (BeginMassUpdate ()) {
foreach (var value in names)
coll.Add (value);
}
